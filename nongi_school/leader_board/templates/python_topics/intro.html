Определения

Каждое занятие будет начинаться с новых определений, которые будут использоваться в курсе.

IDLE (читается айдл)
(Integrated Development and Learning Environment) — это интегрированная среда разработки и обучения на языке Python.
Мы будем использовать IDLE для обучения, так как он включает в себя всё необходимое, но не включает в себя надстройки, необходимые более опытным пользователям, что упростит изначальное использование.
Литерал
Литерал - это простейший способ создания объектов.
Например:
Python 
 1  # числовой литерал
Python 
 "It's a string!"  # строковый литерал
Python 
 [ 1 , 2 , 3 , 4 ]  # литерал списка
Python 
 { 'a' : 1 , 'b' : 2 , 'c' : 3 }  # литерал словаря
Записи выше не имеют смысла сами по себе, с этими литералами можно что-то делать (сравнивать с ними, например) или же присвоить их в качестве значения какой-либо переменной.
Так, например, можно присвоить значения числого литерала к переменной:
Python 
a =  1
Константа
Часто считают, что литерал и константа - это одно и то же, но это не так.
Константа - это некоторое понятное имя, в котором записан литерал. Это дает возможность использовать один и тот же литерал, который может быть не очень удобным в использовании сам по себе. Например:
Python 
NUMBERS =  [ 1 , 2 , 3 , 4 ]
В питоне нет возможности делать значения неизменяемыми, но общепринято называть константы прописными буквами.
Переменные
Изменяемая именованная ссылка на некий объект.
Python 
a =  1 
b =  2 
c = a + b
print ( c)   # выведет 3
Тип данных
Python относится к языкам с неявной сильной динамической типизацией. Неявная типизация - не нужно писать какого типа данных та или иная переменная.
Например, в Java явная типизация и объявление выглядело бы так:
Java 
int a =  1
В Python аналогичная запись будет без указания типа данных:
Python 
a =  1
Динамическая типизация означает, что тип переменной определяется в процессе выполнения программы. То есть одна и та же переменная могут быть разного типа на разных этапах программы. Например, предыдущая программа могла бы продолжиться следующим образом:
Python 
a =  1 
...
a =  'Hello'
Сильная типизация не позволяет проводить операции между различными типами данных.
В Python нельзя производить сложение чисел и бук, хотя в языках со слабой типизацией, например с++, подобные операции доступны.

Типы данных

Встроенные типы данных

Давайте познакомимся с типами данных поближе. Типы данных разделяются на встроенные и не встроенные.
Рассмотрим основные встроенные типы данных, с которыми мы столкнемся в ближайшее время:

  1. None
    Неопределенное значение, ничто.
  2. Логические переменные
    В разных источниках может называться bool.
    Этот тип данных отвечает за истину/ложь.
  3. Числа
    • int - целые числа
    • float - числа с плавающей точкой (дробные)
  4. Строки
    Также называется str. Служит для работы с символьными данными.
  5. Списки
    • list - список
    • tuple - кортеж
  6. Словари
    • Dictionary или сокращенно dict. Используются для хранения парных данных и не только.

Преобразование типов

Для перевода данных из одного типа данных в другой используются специальные ключевые слова. Вы уже могли их видеть - они фигурировали выше. Пример различных переводов:


Python 
a  =  1  # на данный момент это число 
a  =  str ( a )  # мы сделали значение строкой '1' 
a  =  int ( a )  # мы снова сделали значение числовым и оно равно 1 
a  =  float ( a )  # а теперь оно дробное и равно 1.0

В зависимости от того, что мы планируем делать со значением дальше, можно выбрать удобный тип данных для хранения и обработки содержимого.

Первые функции

Подробнее само понятие функции и о том, как создать свою функцию, мы поговорим совсем скоро. Но для того, чтобы сделать свои первые шаги в питоне нам понадобится знание о простейших встроенных функциях: ввода и вывода.

Функция print

Print может принимать одно или несколько значений и выводить их. Например:

Python 
a  =  1 
print ( a )  # вывод будет: "1" 
print ( a , 2 , 'some other data' ) # вывод будет: "1 2 some other data" 
b  =  3 
print ( a , b )  # вывод будет: "1 3"

Немного дополнительной информации о print. В общем случае функция выглядит следующим образом:

print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

Первый параметр: value - значение, которое будет выводиться. В примере выше мы показали, что значений может быть несколько, о чем и говорит знак многоточия.

Следующий параметр: sep=' ' - разделитель (от слова separator), который будет выводиться между значениями функции.По умолчанию этот разделитель равен пробелу, но иногда нужно, чтобы это было иначе. Например:

Python 
print ( 1 , 2 , 3 , sep = '\n'  )  # вывод будет: 
 1 
 2 
 3
